## Launching 

### Production

`python start.py`  

### Development

1. Clone repo
2. Create python virtual environment
   `python -m venv venv`
3. Activate venv  
   `source venv/bin/activate`
4. Install dependencies  
   `pip install -r requirements.txt`

Launching Django server 
`python manage.py runserver 0.0.0.0:8000`
