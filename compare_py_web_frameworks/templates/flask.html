{% trans "Flask is a free open source micro-framework created in 2004. Flask is classified as micro-framework, it doesn't need any special tools or libraries to launch." %}

{% trans "In 2012 one of the biggest social networks for posting pictures Pinterest moved it's API which was originally based on Django to Flask. Pinterest can manage to serve over 10 billions HTTP requests per day only with this software." %}

{% trans "In its basic version, it does not have as many functions as the previously described Django framework. It does not contain a layer responsible for communication with the database or validation of forms. All such functions are offered as extensions to its basic version. The extensions themselves are also developed by the community and people who are not directly in the team of programmers responsible for creating this software." %}

{% trans "Extensions include, among others:" %}

  • {% trans "ORM to communicate with database" %}
  • {% trans "Tools for making database migrations" %}
  • {% trans "Tools supporting different authorization methods" %}
  • {% trans "Form validation" %}

{% trans "The main advantage of the Flask framework is its great freedom in configuration, which translates into the ability to adapt it to the specificity of the problem that developers want to solve. It does not impose the use of specific tools provided with the distribution and apart from providing the basic version with the most necessary functionalities such as rendering HTML pages or routing URLs." %}

{% trans "In it's basic version includes:" %}

  • {% trans "Server and debugger" %}
  • {% trans "Support for unit tests" %}
  • {% trans "HTML templating language Jinja2" %}
  • {% trans "URL dispatcheer" %}
